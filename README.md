# MyAITodo

MyAITodo is a personal AI-enhanced Todo full-stack project focused on practicing the complete workflow of AI product design, web development, API integration, and deployment.

依托于 Codex 桌面版协作开发流程的 AI 全栈练习项目，定位为一个集成 AI 能力的任务管理器。

![MyAITodo Cover](./docs/assets/myaitodo-cover.svg)

## V1 Positioning

V1 is a personal Todo web app for lightweight project builders and task-dense knowledge workers. Users can create and manage tasks, then use AI to:

- rewrite vague task descriptions
- decompose large tasks into executable steps
- get lightweight priority suggestions

## Tech Stack

- Frontend: Next.js + TypeScript + Tailwind CSS
- Backend: FastAPI + SQLAlchemy + Pydantic
- Database: MySQL
- Auth: Email/password + JWT
- Deployment: Vercel + Railway
- AI: OpenAI-compatible API via backend
